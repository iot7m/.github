# IoT7m

**IoT7m** is an open-source project for building and documenting smart home solutions based on **Home Assistant**.  

The project focuses on real-world implementations, custom UI components, and supporting tooling, with an emphasis on local-first setups, explicit configuration, and predictable system behavior.
